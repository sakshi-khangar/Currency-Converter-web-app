# Currency-Converter-web-app
Currency Converter is a simple web-based application that helps users convert money from one currency to another. It is built using HTML, CSS, and JavaScript with a clean and responsive user interface. This project is ideal for beginners to understand JavaScript logic, DOM manipulation, and UI design.

## Features
- Convert currency between USD, INR, EUR, and GBP
- Simple and clean user interface
- Easy to use
- Beginner-friendly project
- Responsive design

## Technologies Used
- HTML – Structure
- CSS – Styling
- JavaScript – Conversion logic

## Conclusion
This Currency Converter project demonstrates how to build a simple and functional web application using HTML, CSS, and JavaScript. It uses a live exchange rate API to provide real-time currency conversion, helping beginners understand API integration, asynchronous JavaScript, and DOM manipulation. The project is easy to use, beginner-friendly, and suitable for learning and practice.
